35//! # `NeuroQuantumDB` Cluster Module
36//!
37//! This crate provides distributed cluster management for `NeuroQuantumDB`,
38//! enabling horizontal scaling through multi-node deployments.
39//!
40//! ## Architecture Overview
41//!
42//! ```text
43//! ┌─────────────────────────────────────────────────────────────┐
44//! │                    NeuroQuantumDB Cluster                    │
45//! ├─────────────────────────────────────────────────────────────┤
46//! │                                                             │
47//! │  ┌─────────────┐    ┌─────────────┐    ┌─────────────┐     │
48//! │  │   Node 1    │    │   Node 2    │    │   Node 3    │     │
49//! │  │  (Leader)   │◄──►│  (Follower) │◄──►│  (Follower) │     │
50//! │  └─────────────┘    └─────────────┘    └─────────────┘     │
51//! │         │                  │                  │             │
52//! │         └──────────────────┼──────────────────┘             │
53//! │                            │                                │
54//! │                    ┌───────▼───────┐                        │
55//! │                    │  Raft Log     │                        │
56//! │                    │  Replication  │                        │
57//! │                    └───────────────┘                        │
58//! │                                                             │
59//! └─────────────────────────────────────────────────────────────┘
60//! ```
61//!
62//! ## Core Components
63//!
64//! - **Raft Consensus**: Leader election and log replication using `openraft`
65//! - **gRPC Transport**: Inter-node communication via `tonic`
66//! - **Consistent Hashing**: Data sharding across nodes
67//! - **Service Discovery**: DNS-based or static node discovery
68//! - **Cluster Manager**: High-level coordination for multi-node deployments
69//! - **Metrics**: Prometheus-compatible metrics for observability
70//!
71//! ## Usage
72//!
73//! ```rust,no_run
74//! use neuroquantum_cluster::{ClusterConfig, ClusterManager};
75//!
76//! #[tokio::main]
77//! async fn main() -> Result<(), Box<dyn std::error::Error>> {
78//!     let config = ClusterConfig::builder()
79//!         .node_id(1)
80//!         .bind_addr("0.0.0.0:9000".parse()?)
81//!         .peers(vec!["node2:9000".into(), "node3:9000".into()])
82//!         .build()?;
83//!
84//!     let manager = ClusterManager::new(config).await?;
85//!     manager.start().await?;
86//!
87//!     // Get cluster status
88//!     let status = manager.status().await;
89//!     println!("Cluster healthy: {}, Leader: {:?}", status.healthy, status.leader_id);
90//!
91//!     // Get Prometheus metrics
92//!     let metrics = manager.metrics().to_prometheus();
93//!     println!("{}", metrics);
94//!
95//!     Ok(())
96//! }
97//! ```
