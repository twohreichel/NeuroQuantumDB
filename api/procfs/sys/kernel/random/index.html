These files provide additional information about the /dev/random device

Note that some of these entries are only documented in random(4), while some are also documented under proc(5)

Functionsยง

boot_id
This is a read-only file containing a 128-bit UUID generated at boot
entropy_avail
This read-only file gives the available entropy, in bits. This will be a number in the range 0 to 4096
poolsize
This file gives the size of the entropy pool
read_wakeup_threshold
This file contains the number of bits of entropy required for waking up processes that sleep waiting for entropy from /dev/random
uuid
This read-only file randomly generates a fresh 128-bit UUID on each read
write_wakeup_threshold
This file contains the number of bits of entropy below which we wake up processes that do a select(2) or poll(2) for write access to /dev/random. These values can be changed by writing to the file.
