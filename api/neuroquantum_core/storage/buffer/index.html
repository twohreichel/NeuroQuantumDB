Buffer Pool Manager for NeuroQuantumDB

Provides intelligent page caching with:

  • LRU/Clock page replacement policies
  • Dirty page tracking
  • Pin/unpin mechanism for concurrent access
  • Background flushing
  • Memory limit enforcement
