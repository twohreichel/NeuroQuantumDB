§Transaction Management System

Complete ACID-compliant transaction management with:

  • Write-Ahead Logging (WAL)
  • Two-Phase Commit Protocol
  • Multi-Version Concurrency Control (MVCC)
  • Deadlock Detection
  • Crash Recovery
