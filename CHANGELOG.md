# Changelog

All notable changes to NeuroQuantumDB will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-01-22

### Features

- Initial release of NeuroQuantumDB
- Neuromorphic computing engine with self-learning capabilities
- Quantum-inspired search algorithms (Grover's algorithm)
- DNA-based data compression
- REST API with JWT authentication
- QSQL query language
- Single-node production deployment
- Cluster mode (beta)
- WASM support for browser integration
- PHP driver for database connectivity
