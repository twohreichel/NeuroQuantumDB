# Glossary

## A

**ACID**
: Atomicity, Consistency, Isolation, Durability — database transaction properties

**AVX2**
: Advanced Vector Extensions 2 — x86 SIMD instruction set

## B

**B+Tree**
: Self-balancing tree data structure for ordered key-value storage

**Buffer Pool**
: In-memory cache for database pages

## D

**DNA Compression**
: Encoding data using quaternary (4-base) system inspired by DNA

## G

**Grover's Algorithm**
: Quantum algorithm providing O(√N) search in unstructured data

## H

**Hebbian Learning**
: "Neurons that fire together wire together" — biological learning rule

## L

**Lateral Inhibition**
: Winner-takes-all mechanism in neural networks

**LRU**
: Least Recently Used — cache eviction policy

## M

**ML-DSA**
: Module-Lattice Digital Signature Algorithm (NIST FIPS 204)

**ML-KEM**
: Module-Lattice Key Encapsulation Mechanism (NIST FIPS 203)

## N

**NEON**
: ARM SIMD instruction set

**Neuromorphic**
: Computing inspired by biological neural systems

## Q

**QSQL**
: NeuroQuantumDB's SQL extension with quantum/neural operations

**QUBO**
: Quadratic Unconstrained Binary Optimization

## S

**SIMD**
: Single Instruction, Multiple Data — parallel processing

**STDP**
: Spike-Timing-Dependent Plasticity — temporal learning rule

**Synaptic**
: Related to connections between neurons

## T

**TFIM**
: Transverse Field Ising Model — quantum physics simulation

## W

**WAL**
: Write-Ahead Logging — crash recovery mechanism
