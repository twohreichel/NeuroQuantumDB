## Module-Übersicht

### neuroquantum-core
- **quantum**: Quantum-Computing Algorithmen
- **synaptic**: Neuromorphe Computing-Funktionen  
- **plasticity**: Lernalgorithmen und synaptische Plastizität
- **dna**: Data Neural Algorithms
- **security**: Zero-Trust Sicherheitsschicht

### neuroquantum-qsql
- **parser**: QSQL Query-Parser
- **optimizer**: Abfrage-Optimierer
- **executor**: Query-Execution Engine
- **natural_language**: Natural Language Processing

### neuroquantum-api
- **handlers**: HTTP Request Handler
- **middleware**: Security & Logging Middleware
- **auth**: Authentifizierungs-System
- **websocket**: Real-time WebSocket Verbindungen
